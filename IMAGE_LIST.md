## 🚀 Quick Download Guide

### Option 1: Manual Download
1. Go to recommended websites
2. Search for each item
3. Download in specified size
4. Rename to exact filename
5. Place in correct folder

### Option 2: Use Placeholder Generator
For testing, create placeholder images:
- https://via.placeholder.com/400x300.png?text=Image+Name
- https://placehold.co/400x300/blue/white?text=Image+Name

### Option 3: Use Free Stock Photo Sites
**Best Sites:**
- ✅ Unsplash.com (High quality, free)
- ✅ Pexels.com (Free stock photos)
- ✅ Pixabay.com (Free images)
- ✅ Flaticon.com (Icons, requires attribution or premium)

---

## 📝 Image Guidelines

**Format:** JPG for photos, PNG for logos/icons
**Size:** Keep images under 500KB each
**Resolution:** Minimum 300x300px, maximum 1920x1080px
**Naming:** Use exact filenames listed above (lowercase, underscore-separated)
